Ejercicio #1

Listado de todos los usuarios registrados en la base de datos.

  • Nombres
  • Apellidos
  • Correo
SELECT nombre, apellidos, correo FROM `usuarios`;

Ejercicio #2

Listado de departamentos registrados en la base de datos.

  • Nombre del departamento
  • Nombre del pais
SELECT d.nombre AS 'Nombre del departamento', p.nombre AS 'Nombre del país' FROM departamentos d JOIN paises p ON d.paises_id = p.id;

Ejercicio #3

Listado de escuelas por un país determinado.

  • Nombre de la escuela
  • Nombre del pais
SELECT e.nombre AS 'nombre de la escuela', p.nombre AS 'nombre del país' FROM escuelas e INNER JOIN paises p ON e.paises_id = p.id WHERE p.nombre = 'españa';

Ejercicio #4

Listado de usuarios ordenados (de mayor edad a menor edad).

  • Nombres
  • Apellidos
  • Fecha de nacimiento
SELECT nombre AS 'nombres', apellidos AS 'Apellidos', fecha de nacimiento AS 'Fecha de nacimiento' FROM usuario ORDER BY fecha de nacimiento ASC;

Ejercicio #5

En que departamento hay más usuarios registrados.

  • Departamento
  • Id
SELECT d.nombre AS 'Departamento', d.id FROM departamentos d INNER JOIN usuario u ON d.id = u.departamentos_id GROUP BY d.nombre, d.id ORDER BY COUNT(u.id) DESC LIMIT 1;

Ejercicio #6

Cual es el país con más usuarios registrados.

  • Pais
  • Id
SELECT p.nombre AS 'País', p.id FROM paises p JOIN departamento d ON p.id = d.paises_id GROUP BY p.nombre, p.id ORDER BY COUNT(d.id) DESC LIMIT 1;

Ejercicio #7

Listar solo 3 departamentos.

  • Nombre del departamento
  • Id
SELECT nombre AS 'nombre departamento', id FROM departamentos LIMIT 3;

Ejercicio #8

Listar la información de un usuario determinado.

  • Nombres
  • Apellidos
  • Correo
  • Nombre del colegio
  • Nombre del departamento
  • Nombre del pais
SELECT u.nombre AS 'nombres', u.apellido AS 'apellidos', u.email AS 'correo', e.nombre AS 'nombre del colegio', d.nombre AS 'nombre del departamento', p.nombre AS 'nombre del país' FROM usuarios u JOIN escuela e ON u.escuela_id = e.id JOIN departamento d ON u.departamentos_id = d.id JOIN paises p ON d.paises_id = p.id WHERE u.id = '11';

Ejercicio #9

Listado de usuarios con un nombre determinado “and” .

  • Nombres
  • Apellidos
SELECT nombre AS'Nombres', apellidos AS 'Apellidos' FROM usuarios WHERE nombres LIKE '%and%' OR apellidos LIKE '%and%';

Ejercicio #10

Cuantos usuarios viven en un departamento determinado

  • Cantidad de usuarios “Numero”
SELECT COUNT(*) AS 'numero' FROM usuarios WHERE departamentos_id = 1;

Ejercicio #11

Listado de usuarios que tengan más de 20 años.

  • Nombres
  • Apellidos
  • Fecha de nacimiento
SELECT nombre AS 'Nombres', apellido AS 'Apellidos', fecha de nacimiento AS 'Fecha de nacimiento' FROM usuarios WHERE fecha de nacimiento <= DATE_SUB(CURDATE(), INTERVAL 20 YEAR) ORDER BY ASC;

Ejercicio #12

Listado de usuarios que tengan entre 20 y 60 años.

  • Nombres
  • Apellidos
  • Nombre capital
SELECT u.nombre AS 'Nombres', u.apellido AS 'Apellidos', p.capital AS 'Nombre capital' FROM usuarios u JOIN departamentos d ON u.departamentos_id = d.id JOIN paises p ON d.paises_id = p.id WHERE u.fecha de nacimiento>= DATE_SUB(CURDATE(), INTERVAL 60 YEAR) AND u.fecha de nacimiento <= DATE_SUB(CURDATE(), INTERVAL 20 YEAR);

Ejercicio #13

Listado de colegios que tengan el numero “7” en su número de teléfono

  • id
  • nombre
  • direccion
  • telefono

Ejercicio #14

Listado de usuarios que no estén asociados a ningún departamento

  • Nombres
  • Apellidos
  • Correo
  • Nombre del departamento
SELECT nombre AS 'Nombres', apellido AS 'Apellidos' FROM usuarios WHERE departamentos_id IS NULL;
